# cumulus
CloudFormation alternative

### Usage

To run cumulus,
```
cumulus <module> <action>
```

For details, run
```bash
cumulus <module> help
```
or
```bash
cumulus help
```

### Dependencies

Cumulus uses [Bundler](http://bundler.io/) to manage dependencies. Run
```bash
gem install bundler
```
followed by
```bash
bundler install
```
to resolve dependencies
